LESSON 5 - USING HTML FORMATTING TAGS

Formatting Tags

1. Bold tag.

Syntax: <b>Text</b>

Example:

This text is <b>bold</b>

Output:

This text is bold

2. Strong tag.

Syntax: <strong>Text</strong>

Example:

This text is <strong>important</strong>

Output:

This text is important

3. Italic tag.

Syntax: <i>Text</i>

Example:

This text is <i>italic</i>

Output:

This text is italic

4. Emphasized tag.

Syntax: <em>Text</em>

Example:

This text is <em>emphasized</em>

Output:

This text is emphasized

5. Superscript tag.

Syntax: <sup>Text</sup>

Example:

This text is <sup>superscripted</sup>

Output:

This text is superscripted

6. Subcscript tag.

Syntax: <sub>Text</sub>

Example:

This text is <sub>subscripted</sub> text

Output:

This is a subscripted text

Here are the other formatting tags that we can use.

Tag Description
<del> It displays as deleted text.
<ins> It displays as inserted text.
<small> It displays text one font size smaller than the rest of the text
<mark> It displays as deleted text

7. Nested tag.

Syntax: <strong><em>Text</em></strong>

Example:

This text is <strong><em>Nested Tags</em></strong>

Output:

This text is Nested Tags
